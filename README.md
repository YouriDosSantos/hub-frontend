# 🎨 Finance Hub — Frontend

A modern, responsive financial CRM frontend built with **React**, **Vite**, **Axios**, and **React Router**. Designed to integrate seamlessly with the Finance Hub backend, delivering secure authentication flows, clean UI components, and a scalable architecture suitable for real‑world enterprise applications.

## 🚀 Overview

The Finance Hub frontend provides the user interface for a financial CRM system. It includes:

- **Secure authentication** (login, registration, token handling)
- **Role‑aware UI** (Admin, Operator, NewUser)
- **Responsive design** for desktop and mobile
- **Reusable components** and clean folder structure
- **API integration** with Axios + backend JWT authentication
- **Modern React patterns** (hooks, context, modular components)

This project demonstrates strong frontend engineering practices with a focus on clarity, maintainability, and real‑world workflows.

## 🧱 Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | React (Vite) |
| Language | JavaScript / JSX |
| Routing | React Router |
| HTTP Client | Axios |
| State | React Hooks + Context |
| Styling | CSS / Responsive Design |
| Build Tool | Vite |
| API | Finance Hub Backend (Spring Boot) |

## 🔐 Authentication Flow

- Login using OAuth2 Password Grant (via backend)
- JWT stored securely in memory / context
- Axios interceptors for authenticated requests
- Automatic redirect on unauthorized access
- Role‑based UI rendering
